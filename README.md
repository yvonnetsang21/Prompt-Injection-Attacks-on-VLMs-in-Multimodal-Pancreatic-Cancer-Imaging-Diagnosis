# Risk and Mitigation of Prompt Injection Attacks on Vision-Language Models in Multimodal Pancreatic Cancer Imaging Diagnosis

多模态胰腺癌影像诊断场景下，对视觉–语言模型面临的提示注入风险及相应缓解策略进行评估；具体设定与结论以论文为准。

本仓库提供论文主图对应的 **R 脚本**（`Data_visualization_code/`）。端到端实验代码在**录用 / 正式发表后**会补全并更新此处。

**依赖：** R ≥ 4.2；`install.packages(c("reshape2","ggplot2","dplyr","ggpubr","ggthemes"))`。脚本读聚合 CSV，路径见各 `.R` 文件。

录用后在此补充 DOI。

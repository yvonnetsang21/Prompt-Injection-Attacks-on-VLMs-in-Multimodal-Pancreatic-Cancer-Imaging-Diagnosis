# 胰腺影像多模态大模型：安全性与鲁棒性研究（公开可视化代码）

本仓库为配合论文/项目投稿而提供的**有限公开材料**：仅包含用于生成主要图表的 **R 可视化脚本**，便于审稿人与读者在获得相应聚合结果数据后复现论文中的统计图风格与作图逻辑。

## 项目内容概述

本研究围绕**胰腺相关医学影像**场景，在**多任务临床推理**设定下，系统评估视觉–语言模型（VLM）在多种实验配置下的表现，并分析**提示注入（prompt injection）**、模态与张数扰动、序列与打乱设置，以及**防御策略**等对模型输出一致性与任务指标的影响。完整问题定义、实验协议、数据集处理规范与结论以正式稿件为准。

## 环境与依赖

- **R**（建议 4.2+）
- R 包：`reshape2`、`ggplot2`、`dplyr`、`ggpubr`、`ggthemes`

在 R 控制台中可安装例如：

```r
install.packages(c("reshape2", "ggplot2", "dplyr", "ggpubr", "ggthemes"))
```

## 数据与运行方式说明

脚本从 **CSV 聚合结果** 读入数据，路径多为**相对于当前工作目录**的文件名或子目录（例如 `Baseline.R` 中的 `*_performance.csv`、`Attack.R` 中的 `case_level_data.csv`，`Defense.R` 中的 `CSV_DIR <- "part2_supervisor_metrics"` 等）。使用前请将稿件/补充材料中提供的**与图对应的聚合表**置于脚本所期望的位置，或在本地按需修改路径常量。

各脚本末尾或中间段落通常包含 `ggsave` 等输出语句；若您的环境缺字库，可能需按系统调整字体设置。
